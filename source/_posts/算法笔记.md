##	双指针

#####	剑指Offer 	05.替换空字符串

​	请实现一个函数，把字符串 `s` 中的每个空格替换成"%20"。

```javascript
/**
 * @param {string} s
 * @return {string}
 */
var replaceSpace = function(s) {
    if (s === '') {
        return ''
    }
    let fast = s.length - 1;
    let slow = s.length - 1;
    for (let i = 0; i < s.length; i ++) {
        if (s[i] === ' ') {
            slow = slow + 2;
        }
    }
    let newArr = new Array(slow);
    while (fast >= 0) {
        if (s[fast] === ' ') {
            newArr[slow] = '0';
            newArr[slow - 1] = '2';
            newArr[slow - 2] = '%';
            slow = slow - 3;  
        } else {
            newArr[slow] = s[fast];
            slow --;
        }
        fast --;
    }
    return newArr.join('');
};
```

#####	翻转字符串里的单词

给你一个字符串 s ，逐个翻转字符串中的所有 单词 。


示例 1：

输入：s = "the sky is blue"
输出："blue is sky the"

```javascript
/**
 * @param {string} s
 * @return {string}
 */
var reverseWords = function(s) {
    var q = [], r = s.length - 1, l = r;
    while (l > -1) {
        while(s[l] !== void 0 && s[l] !== ' ') {
            l--
        }
        l < r && q.push(s.substring(l + 1, r + 1))
        while(s[l] === ' ') l--
        r = l
    }
    return q.join(' ')
};
```

#####	202.快乐数

// 数组记录重复
var isHappy = function (s) {
